# StayHealthy - Online Medical Appointment Booking Platform

### Project Overview
StayHealthy is a front-end web application designed to bridge the gap between patients and healthcare providers in remote and underserved areas. This platform allows users to find doctors, book appointments, and maintain their medical profiles digitally.

### Features
*   **User Authentication:** Secure Sign-Up and Login for patients and doctors.
*   **Doctor Search:** Find medical specialists based on their expertise and location.
*   **Appointment Management:** Real-time booking and cancellation of medical slots.
*   **Notifications:** Application-wide alerts for upcoming appointments.
*   **User Reviews:** Feedback system for consultations with anti-duplicate submission logic.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

### Setup and Installation Instructions

Follow these steps to get the project running on your local machine:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/itv18/stayahealthy-medical-appointment.git](https://github.com/itv18/stayahealthy-medical-appointment.git)
    ```

2.  **Navigate to Project Directory:**
    ```bash
    cd stayahealthy-medical-appointment
    ```

3.  **Install Dependencies:**
    Ensure you have Node.js installed, then run:
    ```bash
    npm install
    ```

4.  **Start Development Server:**
    ```bash
    npm start
    ```
    The application will be available at `http://localhost:3000`.

5.  **Build for Production:**
    To create an optimized build folder:
    ```bash
    npm run build
    ```
